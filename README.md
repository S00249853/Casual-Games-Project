# Casual-Games-Project

Controls: Tapping the left side of the screen makes the player jump, Tapping the right side of the screen makes the player attack. If the player collides with an enemy, they are sent to the start of the level.

For multiplayer, enter the online tab on the main menu. on one device, enter a session name in the create session text box and click the button. A code should be displayed on the top of the screen. on the other device, enter the code into the Join session text box and click join. the game ends when a player reaches the end of the course.
